I decided to focus on the navigation aspect of web development because I think it is extremely crucial for all websites to be easily navigated by users with no experience or knowledge of web development. It should be painfully obvious to the user what the have to click to get where. On this page, I included a couple of common examples about how web pages typically allow for user navigation.

Tutorial on Accordians

Below is an example of how to make one folder in the accordian (like the one I made on this page). Along with the HTML, there are some basic CSS rules that needed to be applied as well, but those are mostly to the developers preference and are interchangable with your own rules.

                <button class="accordian">Accordian Style Folders</button>
                <div class="panel">
                    <Some text inserted here to collapse>

As shown by my example accordian, the accordian file structure is an important and professional way of storing information on a web page. It also allows the user a certain amount of control as to what they are seeing on a web page

Tutorial on Navbars

Below is an example of how to make a navbar (like the one I made on this page). While simple in nature, it is fundamental to the navigation of countless website designs. They are almost universal to use despite the many differences across the various web examples.

                <div class="navbar">
                    <a href="index.html">Back to Home Page</a>
                    <a href="blog1.html">See my First Blog Post!</a>
                    <a>Blog 3: Coming Soon!!</a>
                </div>

As shown by my example, navbars are a clean way to allow the user to transfer between pages. It is clean and easily accessable and understandable for a user. Navbar use also typically allows a user to see which links they have already clicked so that they can ensure they are visiting new or previous sites.
